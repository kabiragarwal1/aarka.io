# aarka.io

A starter implementation for a **global media management platform** that helps small businesses:
- Create campaign content.
- Translate for global channels.
- Publish across social, OTT, display, and AI placements.
- Enforce HIPAA/GDPR-aware compliance checks.
- Auto-manage optimization and analytics loops.

## Quick start

```bash
PYTHONPATH=src python -m pytest -q
```

## What's included
- `src/media_management`: core models, compliance guardrails, and full-cycle orchestration.
- `docs/platform-blueprint.md`: product architecture and roadmap.
- `tests/test_platform.py`: baseline quality checks for create/publish/analyze/optimize flow.
